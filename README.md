# Minimal node starter project

## Packages included

- dotenv
- nodemon
- rimraf
- ts-node
- swc/cli
- swc/core

## Setup

- **`pnpm i`** to install modules

- **`pnpm dev`** to start nodemon

### SWC

- [SWC](swc.rs) is **_rust based_** and can be used for both compilation and bundling
